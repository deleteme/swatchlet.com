About Swatchlet.com

Swatchlet.com was created by Matt Garrett to make it easy to show colors and color swatches to friends.

This site is built using Prototype, Scriptaculous, Github, and John Dyer's 'PhotoShop-like JavaScript Color Picker'.

How to Use Swatchlet.com

  • Paste in any number of hex (FFCC00) color names as a comma separated list after a # following http://swatchlet.com/.
    Example: http://swatchlet.com/#9ADBF9,3182A5,084660,211D13
  • Yubnub.org Example Usage: swatch 9ADBF9,3182A5,084660,211D13
